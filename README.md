# number_to_word

A package for converting numbers to english words, e.g., 

```python
from number_to_word import number_to_word

number_to_word.number_to_word(111)
```

returns

```python
'one hundred eleven'
```

Inputs must be float, int or a castable string, e.g., `'111'`, not `a string`
